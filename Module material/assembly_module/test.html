Helpful Tips

  • Make sure you have all the components for each section before starting
  • Order: Fastener, Washer, Lock Washer, Nut
  • If a hole is too tight for fastener to slide through, use screw driver to loosen it
  • Tighten nuts with fingers first, then use key
  • For each part, put all fasteners on before tightening all of them
