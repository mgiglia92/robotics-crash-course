Mechanical Assembly 2023

Ready to start?

Helpful Tips

  • Make sure you have all the components for each section before starting
  • Order: Fastener, Washer, Lock Washer, Nut
  • If a hole is too tight for fastener to slide through, use screw driver to loosen it
  • Tighten nuts with fingers first, then use key
  • For each part, put all fasteners on before tightening all of them

1. MPU6050

Materials

  • Frame
  • MPU6050
  • 3D printed MPU6050 cover
  • 2x M3x12mm Fastener
  • 2x M3 Hex Nut
  • 2x M3 Lock Washer
  • 2x M3 Washer

2. Caster Wheel

Materials

  • Ball Caster Bag with:
    • Ball
    • Base (with holes for fasteners)
    • 3x Rollers
    • Cover (has snap joints)
  • 3D printed caster spacer
  • 3x M3x30mm Fastener
  • 3x M3 Hex Nut
  • 3x M3 Washer

Note:

  • attach base with spacer first
  • no lock washer this time
  • tighten fasteners before attaching ball
  • don't forget the rollers
